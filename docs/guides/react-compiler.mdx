The React Compiler is a build-time tool designed to optimize React applications automatically. This guide will show you how to enable the React Compiler by integrating it with the `@vitejs/plugin-react`.

```bash
npm install -D @vitejs/plugin-react babel-plugin-react-compiler
```

```ts
// waku.config.ts
import { defineConfig } from 'waku/config';
import react from '@vitejs/plugin-react'; // make sure v5.0.1 or above is installed

const ReactCompilerConfig = {};

export default defineConfig({
  vite: {
    plugins: [
      // cf. https://github.com/vitejs/vite-plugin-react/pull/537
      react({
        babel: {
          plugins: ['babel-plugin-react-compiler', ReactCompilerConfig],
        },
      }),
    ],
  },
});
```

Check out [a working example](https://github.com/wakujs/waku/tree/main/examples/05_compiler), or visit the [StackBlitz demo](https://stackblitz.com/github/wakujs/waku/tree/main/examples/05_compiler) with Waku v0.21.12 and up.
